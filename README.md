📌 Project Overview

This project is a structured digital portfolio demonstrating the completion of five key tasks focused on digital literacy, technical skills, and online safety awareness. It combines practical learning with real-world applications to build both academic and professional competencies.

📂 Project Structure

The repository is organized into the following sections:

🔹 Task 1 – Presentation
Created using Canva/Prezi
Covers digital literacy concepts, tools, and safety practices
Focus on clear design and visual communication
🔹 Task 2 – Portfolio Development
Includes profiles and reflections
Demonstrates understanding of personal digital presence
🔹 Task 3 – Platform Usage
Platforms used: GitHub, LinkedIn, Kaggle
Shows activity, learning progress, and observations
🔹 Task 4 – Email Etiquette
Sample professional emails
Checklist for proper communication practices
🔹 Task 5 – Cybercrime Awareness
Case study on ransomware
“Stay Safe Online” prevention checklist
💡 Key Learnings
Developed strong programming and problem-solving skills through HackerRank
Improved data collection and analysis using Google Forms
Built a professional presence using GitHub and LinkedIn
Understood the importance of clear digital communication
Gained awareness of cybersecurity threats and safe online behavior

⚠️ Key Insight on Cyber Safety

One major takeaway from this project is how small mistakes—like clicking unknown links—can lead to serious cyberattacks such as ransomware. This highlights the importance of staying alert and verifying sources before taking action.
